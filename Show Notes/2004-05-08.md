Qtile for Xorg Install
======================

Assuming you already have a working version of xorg packages.  The following will be required for you to install Qtile on Debian Bookworm.

```
sudo apt install \
    python3 \
    python3-pip \
    python3-venv \
    python3-v-sim \
    python-dbus-dev \
    libpangocairo-1.0-0 \
    python3-cairocffi \
    python3-xfcffib \
    libxkbcommon-dev \
    libxkbcommon-v11-dev \
    git \
    sxhkd

```
